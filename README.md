#     NeXus

Sistema de automatización en PowerShell para instalación y configuración de aplicaciones en Windows.

---

##     Descripción

NeXus es una herramienta diseñada para automatizar procesos repetitivos en sistemas Windows, facilitando la instalación, configuración y ejecución de tareas mediante scripts optimizados.

Este proyecto fue desarrollado de manera independiente con enfoque en eficiencia, organización y escalabilidad.

---

##     Características

- Automatización de instalación de aplicaciones
- Configuración rápida de entorno
- Ejecución de scripts en un solo comando
- Estructura preparada para expansión modular
