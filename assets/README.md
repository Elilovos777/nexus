# 🎨 Assets — NeXus

Esta carpeta contiene recursos visuales del proyecto.

## Contenido actual

```
assets/
├── README.md              # Este archivo
└── screenshots/           # Capturas de pantalla (añadir manualmente)
    ├── splash.png         # Pantalla de inicio con efecto glitch
    ├── instalar.png       # Vista de instalación de apps
    ├── mis-apps.png       # Vista "Mis Apps" con botones ▶
    ├── hardware.png       # Escáner de hardware y compatibilidad
    ├── tweaks.png         # Panel de optimizaciones
    └── seguridad.png      # Limpieza de seguridad profunda
```

## Cómo tomar capturas para el repositorio

1. Ejecuta NeXus con el comando de instalación del README
2. Navega a cada sección
3. Usa `Win + Shift + S` para capturar cada vista
4. Guarda los archivos en `assets/screenshots/` con los nombres indicados
